# Requirements
Ruby `3.1.3` is necessary for this. Ruby `3.2.0` presents breaking changes that makes it incompatible. 

# Commands 
- To install all dependencies: `bundle install`
- Run local server: `bundle exec jekyll serve --baseurl=""`
